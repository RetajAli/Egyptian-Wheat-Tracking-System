# Documentation

This folder contains the project documentation for the Egyptian Wheat Tracking System.

## Contents

- Project Report
- System Description
- Design Pattern Explanations
- Pattern Justification
- Wheat Supply Chain Workflow
- UML Design

## Description

The documentation explains the overall system architecture, wheat tracking workflow, and the software design patterns used throughout the project.

The report describes how wheat is monitored from distribution centers through transportation, storage, processing, and final production stages while ensuring quality and quantity control.
